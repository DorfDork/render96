**!DEPENDENCIES!**

This mod requires Character Select. If you do not currently have Character Select, you can download it here:

[https://github.com/Squishy6094/character-select-coop/releases](https://github.com/Squishy6094/character-select-coop/releases)



**!HOW TO INSTALL!**

If you do not have a build of SM64 Coop Deluxe yet, you can download a build at [https://sm64coopdx.com](https://sm64coopdx.com) .

* In the sm64coopdx folder you've downloaded, you'll see a folder named "mods". Extract the contents of the .zip file directly into the "mods" folder.
* After that, open the .exe file, and (if it is your first time) follow the instructions therein, until you reach the title screen.
* Click Host, then check your mods list. You should now see Render96 A Mod Pack. Make sure that box is toggled to On, along with Character Select.
* Press back, then click Host.

You're now ingame! Have fun playing!
